Parâmetros Flow AI

Os ajustes técnicos que separam uma imagem gerada de uma obra de arte comercial.

O Flow AI, baseado na tecnologia Google, possui controles específicos que nos permitem ajustar a composição final sem alterar o prompt principal. Abaixo, traduzimos os principais parâmetros para o nosso fluxo.

Aspect Ratio (Formato)

O formato define onde a imagem será usada. Escolher o aspect ratio correto no início evita cortes indesejados pela IA.

Uso Comando/Config Aspect Ratio
Instagram Reels / TikTok Vertical 9:16
Post Feed / Facebook Square 1:1
Banner Site / YouTube Landscape 16:9

Prompt Negativo (O que Omitir)

O Flow AI é muito sensível a marcas d'água e textos. Sempre que usar o modo Expert, garanta que seu campo de prompt negativo contenha termos de "limpeza":

text, watermark, blurry, low quality, distorted hands, letters, signatures, unrealistic, oversaturated colors.

Comparação: Midjourney vs. Flow AI

Se você veio do Midjourney, aqui estão os equivalentes técnicos para manter seu nível de produção:

  • --ar no MJ ➔ Configuração de "Size" no Flow.
  • --stylize no MJ ➔ Configuração de "Creativity" no Flow.
  • --v 6.0 no MJ ➔ Ativar "High Fidelity Mode" no Flow.
Nota Técnica: Diferente de outras IAs, o Flow AI entende melhor linguagem natural do que códigos matemáticos. Prefira "shot in 8k resolution" do que apenas números isolados.
